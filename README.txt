# ProjetoBD1
Repositorio Projeto Banco de dados 1 

Para utilizar o sistema siga os seguintes passos:
 
1:  Faça o download do software XAMPP pelo link: https://www.apachefriends.org/pt_br/download.html e instale em sua máquina;
2:  Abra a aplicação do XAMPP e de START no Apache e MySQL;
3:  Abra o PHPMYADMIN pelo endereço: http://localhost/phpmyadmin;
4:  Crie um novo banco de dados nomeado como 'bd1' importe nele o arquivo SQL 'bd1' do reposirório;
5:  No diretório onde foi instalado o XAMPP, substitua o diretório 'htdocs' pelo 'htdocs' do repositório;
6:  Começe a utilizar o sistema com o endereço: http://localhost/SipEm/index.php.
